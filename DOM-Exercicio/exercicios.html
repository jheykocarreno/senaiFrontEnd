2

Favoritar o Card

if/else · style · classList.add · classList.remove
O que fazer: Ao clicar no botão, verifique o valor da variável favoritado (começa como false). Se não estiver favoritado: adicione a classe card-favoritado-ex ao card e mude o texto do botão para "💔 Desfavoritar". Se já estiver favoritado: remova a classe e volte o texto original. Não esqueça de inverter a variável a cada clique.

Breaking Bad

Drama · Série

Clique no botão para favoritar...
💡 Dica Use uma variável let favoritado = false fora da função. Dentro da função: favoritado = !favoritado inverte o valor. Depois use if (favoritado) { ... } else { ... } para decidir o que fazer. Para mudar o texto do botão: document.getElementById('btn-ex2').textContent = '...'

    💡 Dica — passo a passo 1. Leia o .value do input  |  2. Crie um <li> com createElement  |  3. Defina o textContent  |  4. Crie o botão e no onclick use this.parentElement.remove()  |  5. appendChild do botão no li  |  6. appendChild do li no ul
